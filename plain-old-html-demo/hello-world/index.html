'c_fill,h_500,w_500': The section of the URL between 'upload' and 'v1660004846' transforms our image. 'c_fill' resizes our image so that the height is 500 pixels and the width is also 500 pixels. It resizes it before the image is downloaded so the download is much faster than if we took the original image (8192 pixels width and height) and resized it in our browser.
